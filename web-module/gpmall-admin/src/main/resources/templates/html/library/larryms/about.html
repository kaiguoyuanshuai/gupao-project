关于LarryMS框架核心组件说明

LarryMS框架作为layui的扩展与应用，提供了自主开发的核心js组件库以支持全端场景的开发，紧贴业务需求，降低学习成本，遵循Layui开箱即用的理念，简洁，高效和极致，借助LarryMS，您可以用于快速开发各种MIS系统，如CMS、OA、CRM、ERP等。优秀，发自至心，品质源于执着，坚持，卓尔不群

补充：随着LarryMS全新上线，将会以VIP社员的需求共性为导向，紧贴业务需求，为用户企业缩短开发周期，提高工作效率，降低开发者上手成本，专属的VIP文档与场景规范示例。

LarryMS核心组件封装了大量基础功能，如第三方jquery插件智能加载，多终端设备智能识别匹配，前端常用工具类，各类验证扩展，高灵活度的右键菜单，Layui tab选项卡的高度扩展，解决传统iframe性能问题，常用表单封装，树形表等等不一一罗列，LarryMS所追求的目的是提高效率，解放前端，降低开发成本与周期，急您所需，供您所求。

使用举例：(详细介绍与使用范例在会员专属文档中查看) 

layui.use(['larryms','larryTab','LarryMenu'],function(){
    var $ = layui.$,
        larryms = layui.larryms,
        larryTab = layui.larryTab,
        LarryMenu = layui.LarryMenu; 
    //菜单初始化
    larryTab.menuSet({
      type: 'GET',
      url: layui.cache.menusUrl,
      data: layui.cache.menuData,
      left_menu: '#larryms_left_menu',
      leftFilter: 'LarrySide'
    });
    larryTab.menu();
    //使用larryms.plugin()加载各个版本的jq插件，有些jq插件使用频率不高但却需要到，可以智能加载，无需担忧jq低版本高版本兼容问题
    //larryms底层已全部为您解决，无效率延迟问题
    larryms.plugin('snap.svg-min',func);//func为您回调函数 可自定义
    //详情可见会员专属文档示例
    //...
});
